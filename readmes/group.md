# G
The G element is a container used to group other elements.  
Attributes are inherited by its children.

Android:  
GS: Software , shadow  (api < 28)
GH: Hardware



```JS
  import { G ,GS, GH } from 'react-native-painter'
    <Painter style={styles.Painter} 
    >
        <G >
             <Line />
            <Circle />
            <Path />
        </G>   
    </Painter>
```
